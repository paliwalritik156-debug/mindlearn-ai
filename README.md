# 🎓 MindLearn AI – AI Study Assistant

## 📖 Overview

MindLearn AI is an AI-powered study assistant designed to help students learn smarter and faster using Generative AI. The platform enables intelligent document analysis, contextual question answering, personalized learning support, and interactive AI conversations to enhance the overall study experience.

This project combines modern AI technologies with an intuitive user interface to create a powerful educational companion for students.

---

## ✨ Features

* 📄 PDF and document analysis
* 🤖 AI-powered question answering
* 💬 Interactive chat-based learning
* 🧠 Personalized study assistance
* 📚 Context-aware responses
* 📝 Chat history management
* ⚡ Fast and responsive user interface
* 🎯 Student-focused learning experience

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/paliwalritik156-debug/mindlearn-ai.git
cd mindlearn-ai
```

### Install Dependencies

```bash
pip install -r requirements.txt
npm install
```

### Run Application

```bash
streamlit run code.py
```

or

```bash
npm run dev
```
